# leo-lov

Opgaven leo-lov går ud på at bruge React-biblioteket til at bygge en hjemmeside til et fiktivt advokatfirma, Leo-Lov.

Til at fuldføre opgaven, valgte jeg at bruge: 
* HTML-snippets og CSS-styling fra "La Cuisine", en tidligere opgave fra grundforløbet (nav, header og main). (Refactoring af HTML til JSX.)
* HTML-snippets og CSS-styling fra "BrianAir", grundforløbets afslutningsprøve (den globale footer). (Refactoring af HTML til JSX.)
* React-snippets fra mine samlede [react-pros-and-cons](https://github.com/male9-techcollege/react-pros-and-cons) opgaver. 

Opgavens afleveringsadresse er:
https://github.com/male9-techcollege/leo-lov
